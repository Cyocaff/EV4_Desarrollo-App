Instrucciones de uso:

Debido a que esto usa cosas de servidor
para abrirlo de forma correcta hay que estar en este mismo directorio
y ejecutar en la terminal:

python -m http.server

una vez hosteado en el PC se puede acceder al archivo en la red local
preferiblemente se debe acceder desde el navegador del celular ya que 
esto esta pensado para ser una app para celular. 
(Igualmente es bueno acceder en PC para testear funciones o leer errores en consola)

Para conectarse el link deberia ser:
Ejemplo de IP:
http://111.111.111.1:8000/index.html
En pocas otras palabras: IP:8000/index.html

Para conectarse a otros archivos por ejemplo otros HTML en carpetas hay que usar algo como
http://111.111.111.1:8000/script-modules/temp-html/admin-main.html

A veces el servidor no se actualiza, para arreglarlo hay que pulsar
CTRL + C en terminal para terminar el proceso y volver a ejecturarlo.
